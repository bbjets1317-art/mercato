# Stock Portfolio Tracker

A Python-based stock market portfolio tracker with S&P 500 analysis and performance metrics.

## Features
- Portfolio tracking and analysis
- S&P 500 market data integration
- Performance metrics and visualization
- Real-time stock price updates

## Tech Stack
- **Backend:** Python
- **Frontend:** JavaScript, HTML/CSS
- **Data:** Market data APIs

## Getting Started

1. Clone the repository
```bash
git clone https://github.com/bbjets1317-art/stock-portfolio-tracker.git
cd stock-portfolio-tracker
```

2. Install dependencies
```bash
pip install -r requirements.txt
```

3. Run the application
```bash
python mercato_app.py
```

## Project Structure
- `mercato_app.py` - Main application file
- `mercato-ui/` - Frontend interface
- `requirements.txt` - Python dependencies
